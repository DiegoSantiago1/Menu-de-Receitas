# 🍽️ Buscador de Receitas

Aplicação web simples que permite buscar receitas a partir de um ingrediente.
Os resultados são exibidos em forma de cards e, ao clicar em uma receita, é possível visualizar todos os detalhes.

---

## 🚀 Funcionalidades

* 🔍 Buscar receitas por ingrediente
* 📋 Exibir lista de receitas encontradas
* 📖 Visualizar detalhes completos da receita
* 🧾 Lista de ingredientes e medidas
* ▶️ Link para vídeo da receita
* ⏳ Indicador de carregamento durante a busca
* ⚠️ Tratamento de erros (API ou conexão)

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* API externa: TheMealDB

---

## 🌐 Como funciona

1. O usuário digita um ingrediente no campo de busca
2. A aplicação faz uma requisição para a API
3. As receitas são exibidas na tela
4. Ao clicar em uma receita, os detalhes são carregados dinamicamente
